## 🚀 What is Fable?

Fable turns plain English descriptions into beautiful, high-fidelity UI mockups in seconds — powered by Google Gemini AI.

No Figma. No drag and drop. No design experience needed.
Just describe your UI and watch Fable bring it to life.

## 🌌 How It Works

1. User types a natural language UI description
2. Fable sends the prompt to Google Gemini API
3. Gemini returns a structured JSON array of UI components
4. Fable renders them as a clean SVG mockup in the browser
5. User can download the mockup as an SVG file
